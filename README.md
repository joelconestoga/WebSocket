# WebSocket
Client/Server WebSocket

Highlights
------

* Upload text, image or video to server
* List all uploaded files on server
* No HTTP, all communication/authentication through WebSocket
* Java Server
* JavaScript Client (Browser)
* Android Client (Mobile)
